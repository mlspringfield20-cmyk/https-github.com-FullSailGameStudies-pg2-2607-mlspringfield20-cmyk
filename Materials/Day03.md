# 📘 Day 03 Lecture Practices

## 💻 Default Parameter Coding Practice

### 🧩 Part C-1.1: add a default value
1. Open `Day3.h`
2. Modify the ShiftColors method.
   - Give the channelToShift parameter a default value of RED. 

### 🧩 Part C-1.2: call the method

1. Open `Day3.cpp` and find the comment `Part C-1.2`
2. In the PartC_1() method in Day3.cpp write an if-else.
   - If the channel is red, then call ShiftColors but don't pass the channel variable.
   - Else call ShiftColors and pass the channel variable.

## 💻 Copying Vectors Coding Practice

### 🧩 Part C-2.1: copy the vector
1. Open `Day3.cpp` and find the comment `Part C-2.1` in the PartC_2() method
2. Copy the ‘lights’ vector to the other vector variables after the call to MakeLights.
3. Call RemoveLights for each of the 'no' vectors. EX: call RemoveLights on noGreens.
   - Make sure the second parameter uses the color you want to remove.
   
### 🧩 Part C-2.2: copy the vector
1. Open `Day3.cpp` and find the comment `Part C-2.2` in the PartC_2() method
2. Add code to each of the switch cases to copy the correct vector to the current variable
